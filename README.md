## Portfolio Structure

### Analytical & Applied Projects (MLB Orgs)

Projects developed for MLB team applications and research questionnaires:

- **Catcher Framing Prediction** - Machine learning model to predict catcher framing probability using pitch-level data. Includes production-ready code and model deployment pipeline.
  
- **Predicting Pitch Velocity Differential** - Analysis and modeling of pitch velocity differentials to understand pitcher performance and pitch effectiveness.

- **Postgame Pitch Analysis App** - Interactive Streamlit dashboard for analyzing pitcher performance from single-game pitch tracking data. Features include:
  - Pitch mix analysis with usage percentages
  - Movement profiles (horizontal vs vertical break)
  - Pitch location plots with strike zone overlay
  - Interactive filtering by outcome, batter handedness, and count situation

### Statistical Modeling & EDA Projects

- **Statistical Models for Data Science** - Advanced statistical modeling project demonstrating expertise in R and statistical analysis techniques.

### Big Data Analysis

- **Big Data Repository Analysis** - Large-scale data analysis project showcasing skills in handling and analyzing big datasets.

### UCSB Baseball Internship Work

- **JUCO Player Analysis** - Player evaluation and categorization work from internship with UCSB Baseball, including analysis of hitters and pitchers with school affiliations.
